# 项目技术约定

> 本仓库包含 `realms`（Go 单体服务）与知识库（`helloagents/`），用于落地 OpenAI 风格 API 中转（proxy/relay，含 Codex/OpenAI 上游接入）并沉淀可验证的设计与实现笔记。

---

## 技术栈

- **当前形态:** 代码 + 文档
- **实现语言:** Go（`net/http` 优先），单服务 `realms`
- **持久化存储（已确认）:** MySQL（用户 Token/Session 仅存 hash；上游凭据明文入库；旧加密格式已禁用）

---

## 开发约定

- **编码:** UTF-8（无 BOM）
- **文档路径:** 所有知识库文档统一存放在 `helloagents/` 下（SSOT）
- **命名约定:** 文件名使用小写英文字母 + 连字符/下划线，避免空格与中文路径

---

## 安全与合规

- **凭据处理:** 任何 Token/Key 只允许以占位符形式出现在文档中，禁止提交真实密钥。
- **合规底线:** 不实现/不记录明显绕过平台规则的做法；涉及登录/授权需遵循官方条款与安全建议。

---

## 测试与流程

- **测试:** `go test ./...`
- **提交:** 当前不强制；建议采用语义化的变更说明（feat/fix/docs/chore）

## 运行（本地）

- 配置示例：`.env.example`
- 启动入口：`cmd/realms/main.go`
- 开发热重载：`make dev`（仅启动本地 air；不自动启动 Docker）
