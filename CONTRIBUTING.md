# 贡献指南（Contributing）

感谢你愿意参与改进本项目。为了减少沟通成本，请在提交变更前先阅读本文档。

---

## 开发环境

- Go 1.22+
- MySQL 8.x（或兼容）

本项目提供本地开发快捷方式：

- 启动服务：`go run ./cmd/realms -config config.yaml`
- 开发热重载：`make dev`
- 运行测试：`go test ./...`

> 提示：`config.yaml` 可能包含敏感信息，默认在 `.gitignore` 中忽略。请从 `config.example.yaml` 复制生成。

---

## 变更类型与建议

- **bug 修复**：请附带复现步骤与修复后的验证方式（最好包含测试）。
- **新功能/行为变更**：请先说明目标、边界与兼容性影响（尤其是 API 行为与计费/用量口径）。
- **文档变更**：以代码行为为准更新文档，避免“文档看起来对但实际不一致”。

---

## 代码风格与质量门槛

- 运行 `gofmt`（必要时）
- 运行 `go test ./...`
- 不要在日志/文档/示例中提交真实 token/key/邮箱等敏感信息

---

## 提交与合并流程（建议）

1. 描述问题与动机（为什么改）
2. 描述改动（改了什么）
3. 给出验证方式（怎么验证）

如你的变更影响了公共行为（API/配置/安全边界），请同步更新对应文档：

- `README.md`
- `helloagents/wiki/*`（知识库索引与说明）
